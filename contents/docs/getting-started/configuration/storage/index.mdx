## Configuring Storage

<Stepper>
  <StepperItem title="Go to your Supabase Project">
Open the dashboard of your Supabase project.
  </StepperItem>
  <StepperItem title="Navigate to Storage â†’ Create a New Bucket">
   Click `New bucket` and create one with a descriptive name, for example: `public`.
  </StepperItem>
  <StepperItem title="Make sure to uncheck Private bucket if you want files to be publicly accessible.">
   You can later add RLS (Row Level Security) or signed URLs for protected access if needed.
  </StepperItem>
  <StepperItem title="Add this bucket to the code">
   Open `/app/api/upload/route.ts` file and change the bucket name in the following line with your bucket name (public):

   ```javascript:route.ts {2}
const { error } = await supabaseAdmin.storage
    .from('neosaas')
    .upload(`public/${fileName}`, buffer, {
      contentType,
      upsert: true,
      cacheControl: '3600',
    })
```
  </StepperItem>
</Stepper>
