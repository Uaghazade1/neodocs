## Prerequisites

To get started with NeoSaaS, ensure you have the following prerequisites installed:

- **Node.js** 18.x or later
- **NPM**
- **Stripe** account for accepting payments

To set up **NeoSaaS**, first buy the boilerplate from <a href="https://neosaas.dev">**this**</a> link. 


Then, using the access you've been given, go to the **Releases** section of the GitHub repository and download the latest version to your computer.

## Run application

<Note type="warning" title="Warning">
To get the required `.env` keys for NeoSaaS to run properly, check the <a href="/docs/getting-started/configuration/"> **section**</a> in the left menu to configure this keys and database.

</Note>

To get started, you can clone the AriaDocs repository directly from GitHub. 
<Stepper>
  <StepperItem title="Unzip the downloaded `.zip` file into a folder of your choice.  
">
   
  </StepperItem>

  <StepperItem title="Next, open the project folder in your favorite IDE (Cursor, Windsurf, VS Code, etc.).
">
  
  </StepperItem>

  <StepperItem title="Copy the .env.example file and rename it to .env">
   After opening the project, copy the `.env.example` file and rename it to `.env` with just copy paste or with this code in terminal.  


    ```plaintext
    cp .env.example .env
    ```
  </StepperItem>

  <StepperItem title="Replace the placeholder values inside `.env` with your actual environment-specific credentials.
">
   ```javascript:.env showLineNumbers
# Change this lines with your actual keys

# Your Supabase keys
NEXT_PUBLIC_SUPABASE_URL="your_supabase_url"
NEXT_PUBLIC_SUPABASE_ANON_KEY="your_supabase_anon_key"

SUPABASE_URL="your_supabase_url"
SUPABASE_SERVICE_ROLE_KEY="your_supabase_service_role_key"

# Your website url without "/" in the end
NEXT_PUBLIC_APP_URL="http://localhost:3000"

# Your Stripe test keys. Don't forget to use live keys on production
STRIPE_SECRET_KEY="sk_test_..."
STRIPE_WEBHOOK_SECRET="whsec_..."

```

  </StepperItem>
  <StepperItem title="Install dependencies">
Run `npm install` on terminal to install all dependencies.



    ```plaintext
    npm install
    ```
  </StepperItem>

   <StepperItem title="Start the application">
Finally, start the development server with `npm run dev` to begin testing the project locally.

    ```plaintext
    npm run dev
    ```

  
  </StepperItem>
</Stepper>
